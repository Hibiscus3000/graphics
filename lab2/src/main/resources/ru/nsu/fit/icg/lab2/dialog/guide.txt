Чтобы открыть изображение нужно нажать либо на соответствующую залипающую кнопку
на панели инструментов, либо на соответствующий пункт меню "Файл" (поддерживаются форматы .jpg, .jpeg, .png, .gif, .bmp).
После открытия изображения над ним можно выполнять следующие операции:
1) Применить один из следующих фильтров: черно-белый фильтр, инверсия, гамма-коррекция,
фильтры выделения границ: Робертса и Собеля, дизеринг: Флойда-Стейнберга и упорядоченный,
медианный фильтр, фильтр сглаживания по Гауссу, фильтр увеличения резкости, фильтры движения
и радиального размытия, а также акварилизация, для этого необходимо выбрать один из фильтров
с помощью залипающих кнопок на панели инструментов, либо с помощью пункта "Использовать" меню
соответствующего инструмента, после чего клик левой кнопкой мыши по изображению будет
менять состояние изображение между оригинальным и отфильтрованным.
Параметры многих из вышеперечисленных фильтров можно настраивать, для этого нужно нажать
либо правой кнопкой мыши по залипающей кнопке на панели инструментов соответствующего инструмента,
либо на пункт "Параметры" в меню соответствующего инструмента.
2) Можно подгонять размер изображения под текущий размер рабочей области, для этого можно использовать
разные режимы интерполяции: Билинейную, Бикубическую и интерполяцию методом ближайшего соседа.
3) Можно изменять размер изображения, используя колесико мыши, при это курсор мыши должен находиться
внутри рабочей области приложения.
4) Можно поворачивать изображение.
