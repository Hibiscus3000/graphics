Доступны следующие инструменты рисования:
1) Прямая линия, параметры: ширина линии.
2) Прозвольная кривая линия, параметры: ширина линии.
3) Правильный многоугольник, параметры: количество вершин (от 3 до 16), внешний радиус,
ширина линии, угол поворота относительно центра в градусах.
4) Правильная звезда, параметры: количество вершин (от 3 до 16), внешний радиус, внутренний радиус,
ширина линии, угол поворота относительно центра в градусах.
5) Заливка области
Для каждого из вышеописанных инструментов можно выбрать цвет рисования.

Параметры инструмента рисования можно настроить в специальном диалоге, для этого
нужно либо нажать на пункт "Параметры" в меню "Инструменты", либо нажать правой кнопкой мыши
на залипающую кнопку соотвествующего инструмента в панели инструментов.

Помимо этого некоторые параметры можно изменять не заходя в диалог настройки параметров:
1) Ширину линии можно настроить с помощью колесика мыши с зажатой клавишой Shift.
2) Угол поворота относительно центра можно настроить с помощью колесика мыши с зажатой
клавишой Ctrl.
3) Размер многоугольника (внешний и/или внутренний радиус) можно настраивать с помощью колесика
мыши без зажатия дополнительных клавиш.

Цвета также можно настраивать с помощью пунктов меню и с помощью залипающих кнопок на панели
инструментов, доступен выбор произвольного цвета с помощью пункта "Произвольный цвет" меню
"Цвет" и с помощью залипающей кнопки с разноцветным кольцом на панели инструментов.

Доступны возможности отмены последнего действия (Ctrl Z), восстановление последнего отмененного
действия (Ctrl X), очистки рабочей области.

Доступны возможности открытия файлов формата .bmp, .png, .jpeg, .jpg, .gif и сохранения файлов
в формате .png.
